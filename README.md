## 下载

### Docker

```
docker pull apachecn0/flygon-quant-notes
docker run -tid -p <port>:80 apachecn0/flygon-quant-notes
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-quant-notes
flygon-quant-notes <port>
# 访问 http://localhost:{port} 查看文档
```
